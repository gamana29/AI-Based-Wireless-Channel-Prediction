Wireless communication signals are highly dynamic due to:

Multipath fading: Signals reflect off surfaces causing constructive and destructive interference.
Shadowing and obstacles: Buildings, trees, and objects weaken the signal.
Noise and interference: Thermal noise, other devices, and environmental factors.
User mobility: Moving devices experience rapid signal changes.

Predicting wireless signal strength over time is critical for:

Optimizing wireless network performance (5G/6G, IoT).
Scheduling transmissions efficiently to avoid dropped connections.
Improving Quality of Service (QoS) in wireless applications.
Research and simulation for adaptive communication algorithms.

This project provides a framework to predict signal strength using machine learning and deep learning, demonstrating:

Classical regression (Linear Regression)
Ensemble learning (Random Forest)
Recurrent neural networks (LSTM)

The project consists of 7 key stages:

Stage 1 – Data Generation / Loading

Synthetic Data:

Purpose: Test models without requiring real datasets.
Generated as a noisy sine wave:

time = np.arange(0, 100, 0.1)

Stage 2 – Preprocessing

Sliding Window Technique:

ML models predict next value based on previous N samples.
Window size 
w
w determines how far back the model looks.
def create_dataset(data, window_size=5):
    X, y = [], []
    for i in range(len(data) - window_size):
        X.append(data[i:i+window_size])
        y.append(data[i+window_size])
    return np.array(X), np.array(y)
Example with window_size = 5:
X (input)	y (target)
[s0, s1, s2, s3, s4]	s5

Normalization (for LSTM):

Scales data to [0,1] or standardized values.
Important for deep learning convergence.

Stage 3 – Model Training
A. Linear Regression
Predicts a linear relationship:

	y=β0​+β1​x1​+⋯+βn​xn​

Easy baseline.
Implementation:
```bash 
from sklearn.linear_model import LinearRegression
model = LinearRegression()
model.fit(X_train, y_train)
predictions = model.predict(X_test)
```


Pros: Fast, interpretable.
Cons: Cannot capture nonlinear fluctuations.

B. Random Forest Regression
Ensemble of decision trees.
Each tree trained on a subset of data.
Predictions averaged across all trees:
y^​=T1​t=1∑T​treet​(x)
	​
Handles nonlinearities and noisy data.
Implementation:
```bash
from sklearn.ensemble import RandomForestRegressor
model = RandomForestRegressor(n_estimators=100)
model.fit(X_train, y_train)
predictions = model.predict(X_test)
```

Pros: Handles complex patterns, robust to overfitting.
Cons: Slower than linear regression, less interpretable.


C. LSTM (Long Short-Term Memory)
Specialized RNN for sequential data.

Predicts sequential signals considering past history.

Implementation Steps:

Reshape data to [samples, timesteps, features]:
X_train = X_train.reshape((X_train.shape[0], X_train.shape[1], 1))
Build model:
```bash 
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import LSTM, Dense

model = Sequential()
model.add(LSTM(50, input_shape=(window_size, 1)))
model.add(Dense(1))
model.compile(optimizer='adam', loss='mse')
```
